# PolyLife - Fitness & Healthy Lifestyle Platform

A modern, responsive web application for managing workouts, nutrition, and a healthy lifestyle, built with **React** and **Tailwind CSS**.

## ✨ Implemented Features (So Far)

- **Modern Hero Section** with phone number input form and full-screen image
- **Interactive Stats Section** with count-up animation on scroll
- **Why PolyLife Section** with 4 info cards and scroll animation
- **Smart Ecosystem** with 8 microservices arranged in a circular layout with glowing lines
- **Glassmorphism Tooltip** centered on screen when hovering over any microservice
- **Vision Section** with fade-in animation on scroll
- **Professional Footer** with distributed links and contact info
- **Fully RTL support** for Persian language
- **Completely responsive design** for mobile, tablet, and desktop

## 🛠️ Technologies Used

- **React 19** - Interactive components
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Modern utility-first styling
- **React Router** - Route management (pages)
- **Custom Hooks** - Scroll animations and count-up effects
- **Intersection Observer API** - Scroll-triggered animations
